This extension adds a context menu option to calculate and apply anomaly ranks to selected HTTP requests. The anomaly rank is calculated using Burp Suite's built-in ranking utilities and displayed in the notes field of each request, making it easy to identify and prioritize potentially anomalous traffic during security testing.

Features

  • Bulk anomaly ranking: Select multiple requests and apply anomaly ranks to all of them at once
  • Automatic rank calculation: Uses built-in ranking utilities to calculate anomaly scores based on response characteristics
  • Persistent annotations: Anomaly ranks are saved in request notes with timestamps for tracking analysis over time

Usage

  1. Select one or more HTTP requests in any Burp Suite tool (Proxy history, Site map, etc.)
  2. Right-click on the selection and choose "Apply Anomaly Rank" from the context menu
  3. View the calculated anomaly rank in the Notes field of each request, formatted as Anom Rank [timestamp]: [rank]
