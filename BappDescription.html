The Anomaly Rank Right Click Context Menu is a Burp Suite extension that enhances request analysis by providing an option to apply an anomaly rank to selected requests. Once selected, it calculates the anomaly rank for each request and displays the results in the notes section of the request. This extension allows for easier tracking and analysis of anomalous requests within the Burp Suite environment.

Features

This extension focuses on one primary function:

  • Apply Anomaly rank to selected requests: With this feature, you can select multiple requests, right-click, and select the Apply Anomaly Rank option. The extension will calculate the anomaly rank for each request and display the results in the notes section of each request.

Usage

Using the Anomaly Rank Right Click Context Menu extension is straightforward:

  1. Compile the extension using gradlew shadowJar.
  2. Load the extension to Burp Suite from the resulting jar file located in the generated directory: build/libs/ApplyAnomalyRank-x.y.z-all.jar, where x.y.z indicates the build version.
  3. In the Burp Suite, select multiple requests you wish to analyze.
  4. Right-click on the selection and choose the Apply Anomaly Rank option. The extension will then calculate the anomaly rank and detail it in the notes section of each request.
